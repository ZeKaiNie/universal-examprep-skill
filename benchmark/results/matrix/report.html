装了这个备考 skill，AI 真的更少「胡编」吗？——MIT 6.006 实测

测法：同一套金标题（65 题，全部锚定 MIT 6.006 官方讲义/习题），让 3 个模型在 3 种条件下作答——

  • 闭卷：不给任何材料，只靠模型自己的知识。
  • 给全材料：把整门课全文塞进提示里（dump）。
  • skill：只把课件建成知识库，模型按需检索（skill 的防幻觉机制）。
一句话结论：在最弱的 Haiku 上，闭卷正确率 18% → skill 惰性检索 93%；越界题上 skill 弃答 100%（闭卷只有 60%，即闭卷在不会的题上更爱硬编）。「给全材料」臂把整门课塞进提示，频繁撞配额/上下文上限——Haiku 上 65 题仅 2 题跑通，跑通的题正确率与 skill 相当，但这种不可行本身正是 skill 定向检索的实际价值。

⚠️ 诚实声明 Caveats

  • 题量 n=65（每条都跨 3 臂同题对比，最公平）。
  • 判分由 Sonnet 4.6 完成，数值题用程序精确比对，并经人工抽查 16 题与判分结果逐题对照，一致性 Cohen's kappa = 0.875（高度一致），故上表数字可信。判分模型与被测模型同属一个家族，是已知局限。
  • 「给全材料」臂把整门课 dump 进提示，频繁撞订阅配额/上下文上限；其报错答案已从计分中剔除，仅在真答案上计分（样本量见上）。这本身说明 dump 全课在工程上不可行。
  • 幻觉/忠实度以整篇讲义为依据，会把“正确但讲义没写”的展开也算作不忠实——对 grounding 基准是合理口径，但解读时需知晓。

Does the exam-prep skill cut hallucination? A MIT 6.006 benchmark

Method: one gold set (65 items, all grounded in official MIT 6.006 lecture notes / problem sets) answered by 3 models under 3 conditions —

  • Closed-book：no materials, parametric knowledge only.
  • Full-materials：the entire course text dumped into the prompt.
  • Skill：course built into a wiki the model retrieves from on demand (the skill's regime).
Bottom line：On the weakest model (Haiku): closed-book correctness 18% → skill retrieval 93%; on out-of-scope probes the skill abstains 100% vs closed-book 60% (closed-book fabricates more when it doesn't know). The full-materials arm dumps the entire course and keeps hitting quota/context limits — only 2 of 65 Haiku items completed; on items that did complete it matches the skill, but that infeasibility is exactly the point in favour of the skill's targeted retrieval.

Why the full-materials column is sparse: dumping a whole course into one prompt is huge and weaker models can't read it all — Haiku mostly fails to return a usable answer here (only 2/65 succeeded, too few to report) and even Opus/Sonnet completed just 50/27; the table counts only those. By contrast, the skill retrieves only the relevant chapter per question, so prompts stay small and all three models answer reliably — which is exactly the skill's value.

🧪 Hallucination & out-of-scope abstention

Hallucination = share of answers with claims not supported by (or contradicting) the source (lower is better; judged against the full lecture, so it penalizes correct-but-unsourced elaboration). OOS abstention = share of not-covered probes where the model honestly says “not covered” (higher is better).

⚠️ Caveats

  • n=65 items, each answered under all conditions.
  • Judging by Sonnet 4.6; numeric items compared programmatically, and validated by a 16-item human spot-check with Cohen's kappa = 0.875 (high agreement), so the numbers above are trustworthy. Judge and tested models share one family, a known limitation.
  • The full-materials arm dumps the whole course and frequently hits subscription-quota / context limits; its error replies are excluded and it is scored on real answers only (sample sizes above) — which itself shows dumping a whole course is operationally impractical.
  • Faithfulness is judged against the full lecture, so correct-but-unsourced elaboration counts as unfaithful — a reasonable grounding criterion, but worth knowing.
  • This report concerns the skill itself only, independent of any future platform. Data and code are reproducible.
