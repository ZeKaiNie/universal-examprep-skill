MiniOS course notes (fictional, self-authored fixture — NOT real course material).

A Vellum page is eight kilobytes and is the smallest unit the MiniOS memory manager allocates.

A quench flushes all dirty cache lines back to persistent storage so a later crash cannot lose acknowledged writes.

Every Gearbox process is granted a time slice of fifty milliseconds before the scheduler preempts it.

The Marlin database keeps its write-ahead log in a file named marlin.wal and replays it on startup.
