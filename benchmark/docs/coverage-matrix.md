## 读这张表

- **左两列（Tier 0 / Tier 1）大量 ✅/🟡**：结构、schema、来源标注规则、路径安全、ingest 都被零成本测到。
- **Tier 2 整列 ❌**：行为冒烟还不存在——这是最大、最便宜可补的缺口。
- **粗体规律**：凡是「能力 = 一段运行时行为」（惰性加载、出题、画图、提示/跳过、疑难追踪、断点恢复、来源标注输出、无 Python 降级）的，**当前只测了「指令存在」，没测「行为发生」**。
- **多数缺口可用确定性结构断言补齐**（对产物文件断言），无需 LLM 裁判，因此 Tier 2 可以做到既便宜又有意义。详见 [`testing-audit.md`](testing-audit.md) §9。
