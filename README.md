# 🎓 通用期末考试 1天极速备考智能教练 (Universal Exam Cram Coach - LLM Wiki V2.0)

[![Agent: Antigravity](https://img.shields.io/badge/Agent-Antigravity-orange.svg)](#)

这是一个**基于 LLM Wiki 架构重构**、**全科通用**的期末考试极速备考 AI 智能体技能（Agent Skill）V2.0 版本。

只要将本技能导入支持的智能体（如 VS Code 智能体插件、Claude Code、Codex、Cursor、Windsurf、Devin 或网页版 GPTs/Gemini），并提供你想要复习的科目资料，智能体就会化身你的**物理防幻觉私人专属备考教练**，带你在 1 天内突击通关。

---

## 🚀 V2.0 重大更新特性 (What's New)

* **⚡ LLM Wiki 目录结构化加载**：丢弃了原先庞大且容易撑爆上下文的单个 Markdown 答案文件。升级为按章节/阶段的 Wiki 物理切片（`references/wiki/`），Agent 会根据复习进度 **Lazy Load (惰性加载)** 对应章节，**Token 消耗直降 90%**，长对话不再卡顿。
* **🛠️ scripts/ingest.py 一键冷启动**：告别了手动建立题库与 Markdown 文件的繁琐操作。**学生完全不需要理会复杂的 JSON 格式**，只需提供大纲或真题，Agent 将在后台自动解析大纲、拼装 JSON 并通过脚本完成 Wiki 物理切片的切割与进度初始化。
* **🔌 无 Python 环境自动降级**：新版增加了降级执行能力。即使学生电脑里没有安装 Python，Agent 也会无缝切换为“手动写入模式”，利用自身的文件写入功能直接在本地铺设 Wiki 目录，**100% 免配置、零摩擦运行**。
* **🎯 标准真题库 quiz_bank.json 抽题**：测试题由“AI 即兴编造”升级为“标准真题库抽测”，规避了 AI 出无解错题、弱智题的毛病。
* **🏃 测试逃生通道 (Hint & Skip)**：针对测试关卡设计了“查看提示”与“2次答错跳过并归档”机制，防止学生因主观题表述差异或卡壳而被死锁在当前阶段。

## 🎯 计算机 0 基础小白快速上手指南

如果你是个从来没写过代码、甚至没听说过编程的文科/理科学生，请按照下面的“保姆级”步骤开始复习：

### 第一步：准备工作（文件怎么放？）
无论你使用的是哪个 AI 客户端（如 **Cursor**、**VS Code 插件**或 **Claude Code 终端**），本技能都需要放在一个名为 `.agents` 的隐藏文件夹里。

1. **新建你的复习文件夹**：在你的电脑上新建一个文件夹，命名为你的学科名字（比如叫 `计算机组成原理复习`），并用你的 AI 软件（如 Cursor）打开这个文件夹。
2. **下载并解压技能包**：从 GitHub 下载本仓库的 ZIP 压缩包，解压后，你会得到一个叫 `universal-examprep-skill` 的文件夹。
3. **放置到指定目录**：
   * 在你的复习文件夹下，新建一个名为 `.agents` 的文件夹，并在其内部再建一个 `skills` 文件夹。
   * 把解压出的 `universal-examprep-skill` 文件夹，整包拖入到刚才建的 `skills` 目录中。
   * **放置完成后的路径应该是**：`计算机组成原理复习/.agents/skills/universal-examprep-skill/SKILL.md`。

> **💡 找不到以 `.` 开头的文件夹怎么办？**
> * **Windows 用户**：在文件资源管理器顶部，点击“查看” -> “显示” -> 勾选“隐藏的项目”，就能看见和新建以 `.` 开头的文件夹了。
> * **Mac 用户**：在访达（Finder）中按快捷键 `Cmd + Shift + .` 可以一键显示或隐藏这类文件。

---

### 第二步：开启复习（话怎么说？）

用你的 AI 软件打开“计算机组成原理复习”文件夹后，直接给 AI 发送以下指令（**不需要你执行任何脚本或编辑任何文件**）：

> **“教练，这是我的复习大纲/重点截图，请在后台帮我解析并初始化我的备考 Wiki 空间。我的考试科目是【你的科目名称】。”**
> *(然后上传你的大纲图片、截图、或把大纲文字贴在后面)*

AI 接收到后，会在后台默默把大纲转换为结构化文件，自动生成你的专属备考计划。初始化成功后，它会提示你：
> **“同学，你的复习空间已建立成功！这里是你的学习进度表 `study_plan.md`。我们第一阶段复习的内容是：总线与存储，准备好了请对我说：【开始学习】。”**

此时你的复习根目录下会自动多出 `study_plan.md` 等文件。你只需要跟着 AI Coach 的“讲-测-纠”节奏互动，答对题目后它会自动带你进入下一阶段，全程完全“傻瓜式”操作。

## ❓ 常见问题与故障排除

* **Q：我的电脑没有安装 Python，运行 `python scripts/ingest.py` 报错怎么办？**
  * **A**：**完全不用担心！** V2.0 增加了无 Python 环境的自动降级机制。当 Agent 在执行脚本发现你的电脑没有 Python 环境时，它会**自动且静默地切换为“手动写盘模式”**——由 AI 自己直接在本地创建 Wiki 目录并拼写章节文件。用户不会感受到任何差别，依然能够顺利开始复习。
* **Q：为什么测试题总是卡在一道题过不去？**
  * **A**：主观计算题/简答题如果学生答案不够精确，可能会被误判。您可以直接在对话中对 AI 说：“**这道题太难了/这道题我想直接跳过**”。AI 会将该题自动归档至你的 `study_progress.md` 错题本中，直接放行，让你在最后一个阶段统一重温。
