## Student-facing Output

开场确认课程并展示工作区绝对路径后，用一句话提示：“材料处理方式：轻量按需（推荐、默认）/ 完整建库？”用户接受推荐、时间紧迫或没有回答时使用轻量模式。只需解释一次：轻量模式只视觉查看当前阶段最多 8 页、保留带图题的独立题面/答案裁图和学习状态，不生成复习讲义；未完成批次可带原因放弃后重开，已讲完进度不可放弃。完整模式才进行全量结构化建库。这个选择与“只在对话教学 / 生成可打印教材”的输出选择相互独立；轻量下即使已保存可打印偏好，它也会休眠，当前仍只在对话教学。

逐题答案解释方式不作为普通开场问题。进入第二代完整复习讲义前先做宿主能力握手：若宿主的官方能力可验证地提供全新独立子上下文，并能把输入和工具都限制在一道题内，则默认使用内部独立子智能体，只提示一次会额外消耗模型额度和时间；无需接口密钥，也不向外部服务上传。若能力不完整（包括不能限制工具）或无法确认，则使用 `ordinary` 并说明原因。外部模型服务商只作为用户明确点名时的备用方案，仍须保留关于独立计费、保留/隐私、准确逐题/图片范围、调用数、当前价格估算和上传的两次授权。

不要把教学节奏加入开场必答题。用户明确要求逐题推进时，只需说明一次：保存的是 `step_by_step` 偏好，但只有 `full` 且允许提问时才有效；否则偏好保留并休眠，当前使用 `batch`。“继续”只推进路由，不构成已讲完证据。

使用[语言规范](../../../docs/language-policy.md)固定词：当前阶段 / 这题考什么 / 标准答题步骤 / 易错点 / 3分钟速记 / 现在轮到你 / 已记录到错题本 / 必背 / 例题 / 例题解答 / 要点解释 / 错题重做 / 疑难复述 / 已初始化备考空间。

- 🟢 来自资料
- 🟡 AI补充，可能与你老师讲的不完全一致
- ⚠️ AI生成答案，非老师/教材提供

中文模式只写简体中文。`bilingual` 逐块先中文，再紧跟纯英文 `> EN:` 镜像；不另造第三套模板，也不能缺一侧。来源块、回执、笔记与教材同样处理。

实质内容先写 `notebook/`，错题同步 `mistakes/`，再回复 3–5 行摘要和 `完整解答：notebook/chNN.md#<anchor>｜目录：notebook/index.md`。只有进度面板、帮助卡、一次性提示可只在聊天显示。网页端用文本断点；写入失败要明说并在聊天给全文。

开场确认已登记课程并显示工作区绝对路径；不得静默创建默认工作区。
