# `exam-cheatsheet` — `zh` 学生侧文案包

> 仅提供文案；行为事实源是 [skills/exam-cheatsheet/SKILL.md](../../../skills/exam-cheatsheet/SKILL.md)。

每章固定四段：

```text
【必背结论/公式】
- ……（只有智能体补充行标 🟡 AI补充，可能与你老师讲的不完全一致）

【例题】
- 每个重点配一道难题；先显示题面图，否则换题面自足的题。

【例题解答】
- 写清公式、代入和基础过程；无资料答案标 ⚠️ AI生成答案，非老师/教材提供。

【要点解释】
- 同类题如何识别并套用答题框架。
```

真实 `cheatsheet.md` 必须嵌入工作区相对 Markdown 图片；纯路径不算显示。每条末尾链接到 `notebook.py add-entry` 回执打印的真实错题本/笔记本/维基锚点；裸题号不是有效锚。

交付：`考前小抄已编译：cheatsheet.md；指定页数打印版：cheatsheet.pdf。` 可按要求重新压页。无本地浏览器时：`已生成 cheatsheet.html，请打开后按 Ctrl+P → 保存为 PDF。`
